Project Title: Python Fleet Analysis with SQL

Description:
This project analyzes employee and department data using Python, pandas, and SQLite.
The analysis is performed in a Jupyter Notebook and demonstrates data loading,  pandas-based statistic analysis.
and SQL queries, 

Data:
- employees.csv
- departments.csv

The SQLite database is created dynamically by the Python code using the CSV files.
No pre-built database file is required.

Tools Used:
- Python
- pandas
- SQLite
- Jupyter Notebook
